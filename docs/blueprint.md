# **App Name**: REC Online

## Core Features:

- User Authentication: Secure login for students and administrators with role-based access control.
- Course Management: Administrators can create, update, and delete courses.
- Student Enrollment: Administrators can enroll students in courses.
- Progress Tracking: Students can view their progress and grades for each course.
- Announcement System: Broadcast important information to users.
- AI Grammar Checker: Suggest and/or fix grammar mistakes.

## Style Guidelines:

- Primary color: Deep blue (#34495E) to convey trust, stability, and professionalism.
- Background color: Light gray (#ECF0F1), very desaturated, to ensure comfortable readability and focus.
- Accent color: Soft purple (#9B59B6) to signal interactivity in the calm color scheme.
- Body and headline font: 'PT Sans' sans-serif font to promote legibility while expressing modernity with a touch of warmth.
- Use a set of simple, modern icons to make the system user-friendly.
- Responsive layout that works well on all devices.
- Subtle transitions to maintain user engagement without overwhelming them.
